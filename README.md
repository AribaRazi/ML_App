# 📱 ML Kit Object Detection App

This project is an Android application developed as part of the **AI-ML Virtual Internship**.  
It demonstrates **object detection using Google ML Kit**.

---

## 🚀 Features

- 📷 Detect objects from images
- 🧠 Uses Google ML Kit for on-device machine learning
- ⚡ Fast and real-time processing
- 📱 Simple and user-friendly interface

---

## 🛠️ Tech Stack

- **Language:** Kotlin  
- **IDE:** Android Studio  
- **ML Library:** Google ML Kit  
- **Build Tool:** Gradle  

---

## 📂 Project Structure

- `MainActivity.kt` – Main logic for object detection  
- `res/` – UI layouts and resources  
- `AndroidManifest.xml` – App configuration
